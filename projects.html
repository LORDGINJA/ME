Cookie Clicker

This is my second big project in computer programming. To see my first, go check out 3 Page Website

My second project, still 9th grade but second sememester, was a remake of a game called Cookieclicker, which took me about two months. Cookieclicker is exactly what it sounds like: a game in which you click a cookie. This project included HTML5, CSS, and JavaScript. I was allowed only to look at the game and guess the code. As simple as this sounds, I was new to Java and found it kind of difficult. In the end, however, I managed to make a very good copy, looking and working very similarly to the actual game.
